# Joy's Calendar

## 主要功能
	1. 顯示台灣、日本...等國家節日
	2. 顯示農曆、２４節氣...等資訊
	3. 新增編輯刪除記事
	4. 通知提醒
	5. 備份記事
	6. 支援離線使用
## 使用的相關函式庫
    - 狀態管理: flutter_bloc
  	- 網路請求: dio, retrofit (Google Calendar API)
	- 本地儲存: hive, shared_preferences
	- 手機通知: flutter_local_notifications
	- 錯誤收集: firebase_crashlytics
	- 備份檔案: firebase_storage
	- Google/Apple ID登入: firebase_auth
	- 廣告: google_mobile_ads
